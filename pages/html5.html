HTML5 permite cargar videos/audios desde la propia fuente sin la necesidad de hacer un 'embed' desde alguna fuente externa (youtube, por ejemplo)

Esto nos ayuda a eliminar el hot-linking

Entre otras opciones podemos setearle 'autoplay' que indica que sera reproducido cuando se cargue el DOM

HTML5 también nos permite incluir gráficos SVG, muy útil para logotipos (por ejemplo) dado que la imagen vectorial es escalable indepedientemente de las dimensiones con las cuales es presentada.

El grafico SVG a continuación es incrustado directamente en el documento HTML

HTML5 también introduce nuevos tipos para campos para formularios, junto a validaciones para estos.

Esto es importante dado que logramos eliminar las validaciones basadas en Javascript
